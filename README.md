# 🛒 Digitronix E-Commerce Backend

Digitronix هو نظام تجارة إلكترونية متكامل لإدارة المنتجات، التصنيفات، المستخدمين، والطلبات.  
هذا المشروع يمثل الجزء الخلفي (Backend) باستخدام **Node.js + Express + MongoDB** مع دعم التوثيق JWT وإدارة الصلاحيات.

---

## 📂 المميزات الرئيسية

- **إدارة المنتجات (Products)**: إضافة، تعديل، حذف، عرض، مع دعم الصور والأسعار والتخفيضات.
- **إدارة التصنيفات (Categories)**: تنظيم المنتجات حسب التصنيفات.
- **إدارة المستخدمين (Users)**: تسجيل، تسجيل دخول، إدارة الصلاحيات (Admin / User).
- **إدارة الطلبات (Orders)**: إنشاء الطلبات، تتبع حالة الطلب، حساب الإجمالي تلقائيًا.
- **التوثيق (Authentication)**: باستخدام **JWT**.
- **رفع الصور (Image Upload)**: باستخدام `multer`.
- **تصفية وفرز البيانات**.
- **حماية المسارات** للمديرين والمستخدمين.

---

## 🛠️ التقنيات المستخدمة

- **Node.js** + **Express**
- **MongoDB** + **Mongoose**
- **JWT Authentication**
- **bcryptjs** لتشفير كلمات المرور
- **multer** لرفع الصور
- **dotenv** لملفات البيئة
- **cors** لدعم الواجهات الأمامية
